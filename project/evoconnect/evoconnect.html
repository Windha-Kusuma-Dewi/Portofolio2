EVOConnect

Description

EVOConnect is a professional social networking platform developed to help users build work connections, expand their professional networks, and find relevant career opportunities . Users can create profiles, showcase portfolios, share experiences, interact with the community, and search for or post job vacancies. The platform adopts a modern architecture with React JS on the frontend to provide a good user experience. Meanwhile, the backend is built using Golang (Go).

This project was developed by a team of six people, with a structured division of tasks ranging from frontend to backend. Teamwork allowed each part of the system to be organized more neatly.

Feature

  • Authentication Feature (Login)
  • Networking System
  • Job & Vacancy Search
  • Posts
  • Dashboard
  • Notifications
  • Modern UI
JavaScript (React JS) Golang Tailwind CSS
