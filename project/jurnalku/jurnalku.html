Jurnalku

Description

Jurnalku is a smart application designed to help teachers and students monitor and manage skills competencies more effectively, in a structured manner, and in real time. With comprehensive features and a clear user flow, Jurnalku provides an easier, more accurate, and more transparent monitoring process.

In this project, my team of three and I were responsible for converting the Jurnalku website into a mobile version. Our main focus was to ensure that every UI/UX element could be optimally adapted to mobile device screens without reducing functionality or user comfort. This process included simplifying the layout, optimizing navigation, and adjusting components to remain responsive and consistent with the original design.

Through this project, I gained deeper experience in team collaboration and the process of converting designs from web to mobile interfaces.
