Ticketing App

Description

This project involves creating a dummy interface for a ticketing app using Flutter, completely following the UI/UX design provided previously. In the process, I built various pages, including a dashboard, purchase page, payment page, pop-up alerts, and payment receipts.

Besides being a practice for building a UI in Flutter, this project helped me strengthen my understanding of design implementation, efficient widget usage, and maintaining alignment between UI/UX concepts and the resulting display. Although still a dummy without a backend, this app is designed to look clean.
