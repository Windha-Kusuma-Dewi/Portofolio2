HewanKu (Petshop)

Description

HewanKu is a UI/UX design concept for a modern pet shop application. This project focuses on developing a user interface and experience that is friendly, informative, and well-structured. In the design process, I began by analyzing the main user needs, such as ease of searching for products and quick access to important information like prices, descriptions, and product reviews.

The entire design was created using Figma, from mapping out the user flow to designing proportional layouts and creating consistent UI components. Through this project, I gained a deeper understanding of how to structure clear navigation, build effective visual elements, and maintain design consistency across every page. The HewanKu project served as an important exercise for me in developing my UI/UX design skills and translating visual ideas into a functional and modern mobile app interface.
