Webshop

Description

Webshop is a practice project I created to develop my skills in building website layouts using pure HTML and CSS. This project features a simple online store page.

In carrying out this work, I studied and applied several important aspects, including:

  • Structuring the page with basic HTML
  • Layout settings using Flexbox
  • Using simple design concepts

Although it does not yet have interactive features or a backend, this webshop is one of the projects that helped me understand the basics of website design and became the foundation before moving on to more complex work.
